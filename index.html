Face is a Modern ORM for PHP that uses the PHP 5.4 traits

Traits are new PHP OOP structure brought with the version 5.4. Learn more about traits

Why Face and what is the idea ?

Two purposes are hidden behind face : performances and strength. Face tries to be strong, it is very efficient for joining tables and keeping children/parents relations between the entities, in the same time he is still easy to use and fast

Face is easy to install and once your database is ready, you can start a project in less than 10 minutes. That's why it is perfect for little and medium sized projects.

Starting a new project is really fast

  1. 1. Create your database with mysql workbench
  2. 2. Pick your favorite framework from the list (~5 minutes)
  3. 3. Generate your models with the Embryo tool (~5 minutes)
  4. 4. Code your awesome application using Face

Joining table made easy thanks to the query builder

You chose what you join, ease of use and performances are your best friends

// create a PDO object : $pdo = new PDO('mysql:host=localhost;dbname=lemon-trees', 'login', 'pswd'); // Create a query to find some trees in the DB $fQuery = Tree::faceQueryBuilder() ->join("lemons") // trees have lemons ->join("lemons.seeds") // each lemon has some seeds ->join("leafs") // trees have also leafs ->where("~age>10"); // only tree older than 10 // Get the datas from the db $trees = Face\ORM::execute($fQuery, $pdo); echo count($trees); echo count($trees[0]->getLeafs()); echo $trees[0]->getLeafs()[0]->getSize();
